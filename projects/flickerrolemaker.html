Flicker Role Maker

This is a template for ROBLOX Studio which allows you to create role screens that almost look like the ones from Flicker.

Features

- Easy customization using a ModuleScript
- Realistic looking Role Screens and Role Cards
- Color Gradients
- Extra Features

If you are interested, check out the game's page here

How to use


Settings

You can find the Settings script inside ReplicatedStorage.
Inside, you will find multiple categories.

CHARACTERS

In this category, you can register characters, so you can let the game run scripts and animations upon loading everything else.
Use the CHARACTER_NAME template which is found inside TEMPLATES and insert it here to register your character. Replace CHARACTER_NAME with the name of your character. (Use the same name as the model in the Explorer.)
